# Mozo Design System

Design system for [Mozo – The Money Saving Zone](https://mozo.com.au/), built with **Storybook** and **Bootstrap 5**.

## Brand

- **Primary color:** `#11322c` (Mozo forest green)
- All primary buttons, links, and accents use this color via Bootstrap CSS variables.

## Stack

- **React** + **TypeScript**
- **Bootstrap 5.3** (latest) for layout and components
- **Vite** for build
- **Storybook 8** for documentation and development

## Components

Use the sidebar to explore:

- **Button** – Primary, secondary, outline, sizes
- **Card** – Header, body, footer, image
- **Badge** – Labels and pills
- **Input** – Form fields with label, hint, error
- **Alert** – Notifications and messages

All components are built on Bootstrap classes and themed with the Mozo primary color. Import from `@/components` or the specific component path.

## Usage

```tsx
import { Button, Card, Badge, Input, Alert } from '@/components'
```

## Design tokens

See **Mozo / Design Tokens** for colors, typography, and spacing. Tokens are defined in `src/design-tokens.ts` and applied via `src/styles/mozo-theme.css`.
